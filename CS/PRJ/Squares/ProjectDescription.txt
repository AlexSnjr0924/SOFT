Описание проекта "Squares" (измерение площадей геометрических фигур)
--------------------------------------------------------------------
0. ".\Doc\Demands.txt" - Постановка задачи

1. ".\SquareMetric" - Директория с исходными файлами библиотеки "SquareMetric.dll"
1.0. ".\SquareMetric\Properties\AssemblyInfo.cs" - Свойства сборки (C#)
1.1. ".\SquareMetric\BA\MathAccount.cs" - Базовые математические операции
     (извлечение квадратного корня с точностью до типа decimal, расстояние между точками)
1.2. ".\SquareMetric\BO\LineObject.cs" - Математические объекты (1 измерение)
1.3. ".\SquareMetric\BO\SquareObject.cs" - Математические объекты (2 измерения)
1.4. ".\SquareMetric\BL\CalculateFlat.cs" - Математические операции с объектами
     (векторное [скалярное] произведение направленных отрезков, вычисление sin [cos] углов)

2. ".\SquareStart" - Директория с исходными файлами программы "SquareStart.exe" (unit test)
2.0. ".\SquareStart\Properties\AssemblyInfo.cs" - Свойства сборки (C#)
2.1. ".\BI\ShapeInput.cs" - Класс для ввода исходных данных (см.п. 2a)
2.2. ".\Program.cs" - стартовый файл проекта
2.3. "App.config" - файл конфигурации -> ".\Last\SquareStart.exe.config"
2a. ".Input\ShapeCoords.xml" - Входные параметры фигур в формате XML

3. ".\Last" - в этой директориис создается библиотека и исполняемый файл
3.1. ".\Last\SquareMetric.dll" - библиотека для вычислений площадей фигур
3.2. ".\Last\SquareStart.exe" - исполняемый файл (тест)
3.3. ".\Last\SquareStart.exe.config" - конфигурация

В процессе выполнения вычисляются площади и периметры фигур, описанных в файле в п.2a.
